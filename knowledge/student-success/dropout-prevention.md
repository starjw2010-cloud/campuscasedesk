# 중도탈락 위험 관리 지침

## 1. 목적·적용 범위
중도탈락 위험 학생의 위험도 평가와 개입 절차를 정한다.

## 2. 처리 절차
1. 위험도 평가(출석·성적·상담)
2. 위험군 분류
3. 맞춤 개입(상담·튜터링·장학 연계)
4. 효과 모니터링

## 3. 필요 서류
- 위험도평가서
- 상담일지
- 상담동의서

## 4. 판정·승인 기준
| 구분 | 기준 |
|---|---|
| 고위험 | 집중 개입 |
| 중위험 | 정기 모니터링 |

## 5. 예외·유의사항
- 개인정보·상담 동의 범위 내 데이터 활용
