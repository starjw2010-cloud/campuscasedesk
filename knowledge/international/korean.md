# 한국어 능력 요건 안내

## 1. 목적·적용 범위
수학에 필요한 한국어(TOPIK) 요건과 미달 시 지원을 정한다.

## 2. 처리 절차
1. TOPIK 성적 확인
2. 요건 미달자 파악
3. 한국어 수업 연계
4. 수강계획 조정

## 3. 필요 서류
- TOPIK성적표
- 수강계획서

## 4. 판정·승인 기준
| 구분 | 기준 |
|---|---|
| 요건 충족 | 정상 수강 |
| 미달 | 한국어 지원 프로그램 연계 |

## 5. 예외·유의사항
- 전공 수강과 한국어 보충 병행
