# 수강 정정 처리 지침

## 1. 목적·적용 범위
수강 정정 기간 내 신청 처리와 정원·선수과목 기준을 정한다.

## 2. 처리 절차
1. 수강변경원 접수
2. 정원·선수과목 확인
3. 지도교수 승인
4. 정정 반영

## 3. 필요 서류
- 수강변경원
- 지도교수확인서

## 4. 판정·승인 기준
| 구분 | 기준 |
|---|---|
| 정원 내 | 정정 승인 |
| 정원 초과 | 불가/대기 |

## 5. 예외·유의사항
- 정정 기간 외 변경 불가
