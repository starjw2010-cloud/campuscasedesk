# 휴학 처리 규정

## 1. 목적·적용 범위
일반·군·질병 휴학의 신청 요건과 처리 절차를 정한다.

## 2. 처리 절차
1. 휴학원·사유증빙 접수
2. 학적 처리
3. 등록금 처리(이월/환불)
4. 결과 통보

## 3. 필요 서류
- 휴학원
- 사유증빙

## 4. 판정·승인 기준
| 구분 | 기준 |
|---|---|
| 일반 휴학 | 연한 내 허용 |
| 질병/군 | 증빙 시 허용 |

## 5. 예외·유의사항
- 등록 후 휴학은 등록금 처리 규정 적용
