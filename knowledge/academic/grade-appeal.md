# 성적 이의신청 처리 지침

## 1. 목적·적용 범위
성적 공시 후 이의신청의 접수·검토·정정 절차를 정한다.

## 2. 처리 절차
1. 성적공시 후 기한 내 이의신청 접수
2. 담당 교수 근거 검토
3. 정정 사유 시 정정, 아닐 시 사유 통보
4. 결과 통보

## 3. 필요 서류
- 성적이의신청서
- 근거자료

## 4. 판정·승인 기준
| 구분 | 기준 |
|---|---|
| 정정 사유 | 성적 정정 |
| 사유 없음 | 원 성적 유지 |

## 5. 예외·유의사항
- 접수 기한(공시 후 일정일) 경과 시 불수리
