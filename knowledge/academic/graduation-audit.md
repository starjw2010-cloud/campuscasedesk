# 졸업요건 사정 지침

## 1. 목적·적용 범위
졸업 요건(학점·필수·성적) 충족 여부 사정 및 미충족 시 안내를 정한다.

## 2. 처리 절차
1. 졸업사정표 작성
2. 요건 충족 점검
3. 미충족 항목 안내
4. 추가 이수 계획 수립

## 3. 필요 서류
- 졸업사정표
- 성적증명서

## 4. 판정·승인 기준
| 구분 | 기준 |
|---|---|
| 충족 | 졸업 사정 통과 |
| 미충족 | 추가 이수 안내 |

## 5. 예외·유의사항
- 전공·교양·졸업학점 별도 점검
