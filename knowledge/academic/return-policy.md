# 복학 처리 규정

## 1. 목적·적용 범위
복학 신청 요건·등록 절차와 미등록 시 처리를 정한다.

## 2. 처리 절차
1. 복학원 접수
2. 휴학 사유 종료 확인
3. 등록금 납부
4. 학적 복원

## 3. 필요 서류
- 복학원
- 등록금납부확인서

## 4. 판정·승인 기준
| 구분 | 기준 |
|---|---|
| 기한 내 | 정상 복학 |
| 미등록 | 제적 검토 |

## 5. 예외·유의사항
- 수강신청 일정과 연계 안내
