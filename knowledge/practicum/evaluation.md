# 기관 평가서 처리 지침

## 1. 목적·적용 범위
실습 종료 후 기관 평가서 수령·반영 및 지연 시 처리 기준을 정한다.

## 2. 처리 절차
1. 기관에 평가서 요청
2. 수령·검토
3. 성적 반영
4. 지연 시 독촉 및 잠정 처리

## 3. 필요 서류
- 기관평가서
- 실습일지

## 4. 판정·승인 기준
| 구분 | 기준 |
|---|---|
| 정상 수령 | 성적 반영 |
| 지연 | 잠정 보류 후 수령 시 정정 |

## 5. 예외·유의사항
- 기관 미회신 장기화 시 지도교수 직접 평가 보완
