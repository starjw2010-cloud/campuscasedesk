# 현장실습 산재·상해보험 가이드

## 1. 목적·적용 범위
실습 전 보험 의무 가입과 미가입·사고 시 처리 기준을 정한다.

## 2. 처리 절차
1. 실습 시작 전 보험 가입
2. 보험가입증명서 제출
3. 미가입 시 실습 보류
4. 사고 시 경위서·진단서 접수 및 보험 처리

## 3. 필요 서류
- 보험가입증명서
- 실습협약서
- 사고경위서(사고 시)

## 4. 판정·승인 기준
| 구분 | 기준 |
|---|---|
| 가입 완료 | 실습 진행 |
| 미가입 | 실습 불가 — 가입 후 진행 |

## 5. 예외·유의사항
- 사고 발생 시 즉시 기관·지도교수 통보 후 보험 청구
