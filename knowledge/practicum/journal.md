# 실습일지 작성·제출 지침

## 1. 목적·적용 범위
주별 실습일지 작성·제출 기준과 미제출 시 조치를 정한다.

## 2. 처리 절차
1. 주별 실습일지 작성
2. 지도교수 확인
3. 미제출 시 독려 및 마감 안내
4. 최종 평가에 반영

## 3. 필요 서류
- 실습일지
- 출근부

## 4. 판정·승인 기준
| 구분 | 기준 |
|---|---|
| 정상 제출 | 평가 반영 |
| 미제출 | 감점 또는 학점 처리 보류 |

## 5. 예외·유의사항
- 마감 후 미제출 지속 시 출근 관리 지침과 연계
