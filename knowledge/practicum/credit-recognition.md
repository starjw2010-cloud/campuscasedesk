# 현장실습 학점 인정 기준

## 1. 목적·적용 범위
현장실습 시간·평가에 따른 학점 인정 및 중도 종료 시 부분 인정 기준을 정한다.

## 2. 처리 절차
1. 실습시간·실습일지·기관평가서 확인
2. 최소 이수시간 충족 여부 판정
3. 부분 이수 시 학점 산정
4. 성적 처리

## 3. 필요 서류
- 실습일지
- 출근부
- 기관평가서

## 4. 판정·승인 기준
| 구분 | 기준 |
|---|---|
| 정상 이수 | 규정 시간 충족 시 정규 학점 |
| 부분 이수 | 정당 사유 + 일정 비율 충족 시 부분 인정 |

## 5. 예외·유의사항
- 중도 포기 건은 early-withdrawal 규정과 함께 적용
