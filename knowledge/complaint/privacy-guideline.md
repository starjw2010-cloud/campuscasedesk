# 개인정보 보호 지침

## 1. 목적·적용 범위
학생 개인정보의 수집·이용·제3자 제공 원칙과 동의 절차를 정한다.

## 2. 처리 절차
1. 개인정보 처리 목적 고지
2. 동의 수령
3. 최소 수집·목적 내 이용
4. 제3자 제공 시 별도 동의

## 3. 필요 서류
- 개인정보동의서

## 4. 판정·승인 기준
| 구분 | 기준 |
|---|---|
| 동의 범위 내 | 처리 가능 |
| 범위 외 | 처리 불가 |

## 5. 예외·유의사항
- 민감정보(성적·건강 등) 별도 동의, 미동의 시 제공 금지
