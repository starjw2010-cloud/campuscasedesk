# 성적정보 제공 기준

## 1. 목적·적용 범위
성적 등 학생 정보의 제3자(학부모 포함) 제공 가능 범위를 정한다.

## 2. 처리 절차
1. 요청자·대상 확인
2. 본인 동의 여부 확인
3. 동의 시 범위 내 제공, 미동의 시 거절·안내
4. 기록 보존

## 3. 필요 서류
- 개인정보동의서
- 본인확인서류

## 4. 판정·승인 기준
| 구분 | 기준 |
|---|---|
| 본인 동의 있음 | 범위 내 제공 |
| 동의 없음 | 제공 불가 — 절차 안내 |

## 5. 예외·유의사항
- 성인 학생 성적은 본인 동의 없이 학부모 제공 불가
