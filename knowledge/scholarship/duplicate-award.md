# 장학금 중복 수혜 심사 기준

## 1. 목적·적용 범위
교내·국가·외부 장학 동시 수혜 시 중복 여부 판정 및 조정 기준을 정한다.

## 2. 처리 절차
1. 수혜내역확인서로 중복 여부 확인
2. 등록금 초과 수혜분 산정
3. 조정/환수 대상 판정
4. 승인 라인 상신

## 3. 필요 서류
- 장학신청서
- 수혜내역확인서
- 성적증명서

## 4. 판정·승인 기준
| 구분 | 기준 |
|---|---|
| 등록금 이내 | 중복 허용 |
| 등록금 초과 | 초과분 조정·환수 |

## 5. 예외·유의사항
- 성적우수+가계곤란 중복은 규정상 1개로 제한
