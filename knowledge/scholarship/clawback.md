# 장학금 환수 처리 가이드

## 1. 목적·적용 범위
부정 수혜·중복·학적 변동 시 장학금 환수 기준과 절차를 정한다.

## 2. 처리 절차
1. 환수 사유 확인
2. 환수액 산정
3. 환수통지서 발송
4. 납부 확인

## 3. 필요 서류
- 환수통지서
- 수혜내역확인서

## 4. 판정·승인 기준
| 구분 | 기준 |
|---|---|
| 학적 변동 | 일할 환수 |
| 부정 수혜 | 전액 환수 |

## 5. 예외·유의사항
- 분할 납부 신청 가능, 미납 시 성적/증명 발급 제한 검토
